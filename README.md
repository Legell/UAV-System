# UAV-System
Данный проект реализует **защищённую VPN-сеть** и систему передачи **MAVLink-телеметрии** от беспилотного летательного аппарата (БВС) к серверу управления.   Основная цель — надёжный обмен данными между **бортовым компьютером (Raspberry Pi)** и **сервером Headscale**,   с дальнейшей обработкой и отображением данных через **Flask-веб-панель**
